# 📊 Data Profiler Project

### Data Preprocessing and Feature Engineering

---

## 📌 Project Overview

This project focuses on performing **data preprocessing, exploratory data analysis (EDA), and data profiling** on a real-world dataset. The goal is to understand, clean, transform, and analyze data collected from multiple sources and prepare it for machine learning applications.

The project simulates a real-world scenario where data is collected from different formats and combined into a single dataset for analysis.

---

## 🎯 Objective

The main objective of this project is to:

* Understand and explore raw data
* Perform data cleaning and preprocessing
* Analyze customer purchase behavior
* Conduct univariate, bivariate, and multivariate analysis
* Generate a detailed data profiling report
* Identify data quality issues and insights

---

## 📂 Data Sources

The dataset used in this project is collected from multiple sources:

* **CSV File** – Customer purchase data
* **JSON File** – Customer information
* **SQL Database** – Subscription or transactional records
* **API Data** – Additional customer-related information

All these sources are merged using a common key (**Customer_ID**) to create a unified dataset.

---

## ⚙️ Project Workflow

### 🔹 Data Acquisition

* Data was loaded from multiple formats including CSV, JSON, SQL, and API sources
* All datasets were combined into a single structured dataset

---

### 🔹 Data Understanding & Cleaning

* Initial exploration was performed using summary methods
* Missing values and duplicates were identified
* Data cleaning steps included:

  * Handling missing values
  * Removing duplicates
  * Fixing inconsistent data types
  * Dropping irrelevant columns

---

### 🔹 Exploratory Data Analysis (EDA)

#### 📊 Univariate Analysis

* Distribution of key variables such as:

  * Age
  * Income
  * Purchases

#### 📊 Bivariate Analysis

* Relationship between:

  * Gender and Purchases
  * Income and Customer Churn

#### 📊 Multivariate Analysis

* Correlation heatmap of numerical features
* Analysis of relationships between multiple variables

---

### 🔹 Data Profiling

A detailed data profiling report was generated to:

* Identify missing values
* Provide descriptive statistics
* Analyze correlations between variables
* Detect potential data quality issues such as:

  * Duplicate records
  * Skewed distributions
  * Highly correlated features

---

## 📈 Key Insights

* Customer purchasing behavior varies based on demographic features
* Income plays a role in customer retention (churn)
* Certain features show strong correlations, indicating possible dependencies
* Data quality issues such as missing values and duplicates were identified and handled
